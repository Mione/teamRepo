Mongo db useful commands and stuff

Show all collections in the mongo db

db.getCollectionNames()

The equivalent of select * from table

db.tablename.find()

Inserting data to table in bulk

db.userlist.insert([{"user":"mio3","firstname":"marius"},{"user":"mio4","firstname":"mio"}])

inserting a single record is as simple as:

db.userlist.insert({"user":"mio3","firstname":"marius"})
