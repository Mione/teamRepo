Node + Express

Navigate to your test directory or create a new directory wherever you want:

mkdir myIncrediblyCoolNodeApp
cd myIncrediblyCoolNodeApp

Check if node is installed globally:

node -v

If you're getting something like node: unknown command or something you need to follow these instructions:

http://blog.teamtreehouse.com/install-node-js-npm-windows

Otherwise you are good to go. So go ahead and install express

npm install express

If any errors install express globally:

npm install -g express

In that directory create a new js file and name it whatever you want. For example:

> app.js

In your app.js file you'll need to add the following:

var express = require ('express');//you need express so include it in.
var app = express();//initialize an app var with the express object.

app.use(express.static(__dirname + '/static'));//define the path to your static file i.e. an html file. It doesn't have to be /static, you can name the dir however you want.

app.listen(8000);//start the server and listen for connections on localhost:8000 or 127.0.01:8000. You can set whatever port you want :D

Note that if in your static directory you have something different than index.html(like app.html) you will need to access it specifically as localhost:8000/app.html.

So yeah by now going to localhost:8000 should give you the static html you've placed on static

If you don't want static content delivered you can just do:

app.get('/', function(req, res){
res.send('Hello World');
});

instead of this bit:

app.use(express.static(__dirname + '/static'));
